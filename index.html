Project Overview

Designed and implemented a secure enterprise network for a medium-sized business with 50+ employees across three departments (Sales, HR, and Development). The network features VLAN segmentation, access control lists, and secure remote access.

Implementation Details

Network Topology

The network consists of:

  • 3 Layer 3 switches (one per department)
  • 2 Routers (one edge, one internal)
  • 1 Firewall appliance
  • 1 DHCP server
  • 1 File server

VLAN Configuration

Each department was assigned its own VLAN to isolate broadcast domains and improve security: 

Switch(config)# vlan 10
Switch(config-vlan)# name Sales
Switch(config-vlan)# exit
Switch(config)# interface range fa0/1-12
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 10

Access Control Lists

Implemented ACLs to control inter-VLAN communication while allowing necessary traffic: 

Router(config)# access-list 101 permit tcp 192.168.10.0 0.0.0.255 192.168.30.0 0.0.0.255 eq 445
Router(config)# access-list 101 deny ip 192.168.10.0 0.0.0.255 192.168.20.0 0.0.0.255
Router(config)# access-list 101 permit ip any any

Testing & Results

Conducted comprehensive testing including:

  • Ping tests between VLANs to verify ACL enforcement
  • Throughput testing with iPerf
  • Security scanning with Nmap
